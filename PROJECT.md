# Wellisch Academic Group

Elite math tutoring website for serious students in Westchester County, NY.

## Project Overview

**Type:** Landing Page / Marketing Website  
**Stack:** Vite + React + TypeScript + Tailwind CSS + shadcn/ui  
**Status:** Development

## Features

### Landing Page Sections
- **Hero** - Eye-catching intro with "By Application Only" badge
- **Services** - Math Tutoring, Homework Support, Test Prep, Study Skills
- **Session Types** - In-home and online options
- **How It Works** - Application process explanation
- **Tutors** - Meet the tutors section
- **Contact** - Get in touch form
- **Footer** - Links and branding

### Technical Features
- Responsive design (mobile-first)
- Modern UI with shadcn/ui components
- Smooth animations
- SEO optimized meta tags
- Dark mode support (via next-themes)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
